Security Features

  • Password Hashing: Uses PBKDF2 with HMAC-SHA256 (Werkzeug).
  • CSRF Protection: Every form has a unique token (Flask-WTF).
  • SQL Injection Prevention: All queries use SQLAlchemy ORM (parameterized).
  • XSS Protection: Jinja2 autoescaping on all templates.
  • Session Security: HttpOnly, SameSite=Lax cookies.
  • Access Control: Every request checks ownership of data.

Your financial data is stored locally in an encrypted database and never shared with third parties.
